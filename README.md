

# Лаб 3: Парсинг газетных дат
Преобразует даты из 3 разных форматов газет в объекты datetime:

Moscow Times: "Wednesday, October 2, 2002" → %A, %B %d, %Y

Guardian: "Friday, 11.10.13" → %A, %d.%m.%y

Daily News: "Thursday, 18 August 1977" → %A, %d %B %Y

Результат: 2002-10-02, 2013-10-11, 1977-08-18

#Лаб 4: Генератор диапазона дат
date_range(start, end) возвращает список дат или [] при ошибках:

Test '2022-01-01' → '2022-01-03': ['2022-01-01', '2022-01-02', '2022-01-03']
Test '2022-01-03' → '2022-01-01': [] (неверный порядок)
Test '2022-02-30' → '2022-02-31': [] (невалидные даты)

Проверки: формат, порядок, календарная корректность.
